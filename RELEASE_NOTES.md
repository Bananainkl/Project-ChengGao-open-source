# 澄稿 1.20.1

本版本修复在线改写质量检查对日期、数字和引用名称的误判，并让自动统稿获得具体缺失事实。

## 本版本说明

- 日期、数量、百分比、英文专名和引用名称会先规范化空格、全半角、引号及“日／号”等等价写法，再判断是否真正遗漏。
- 本机语音转写中的单个裸数字不再误触事实丢失；带单位的数量和日期仍保持严格检查。
- 首次初稿确实漏掉事实时，第二次统稿会收到具体缺失项并逐项补回。
- 用户报告的真实抖音链接已在安装版完成 1,532 字转写、1,252 字改写稿和 56 个分镜，不再出现原质量检查错误。
- 完整 139 项测试通过。

公开源码不代表提供经过 Apple 公证的安装包；发布 DMG 前仍需完成签名、公证和依赖许可核验。
