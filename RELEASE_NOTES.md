# 澄稿 1.20.3

本版本新增完整短视频文件包输出，让文稿、分镜和已经生成的图片可以一次带走。

## 本版本说明

- “处理结果”和“最近处理”新增“输出”，选择目录后自动新建以内容标题命名的文件夹。
- 文件包固定包含 `改写文稿.md`、`分镜与配图提示词.md` 和 `图片/`，不需要再分别整理素材。
- 分镜文档逐镜保存时间、对应文案、完整提示词和图片状态；图片不完整时仍能先输出文稿和全部分镜。
- 已生成图片按镜头顺序编号复制；同名目录自动使用“标题 (2)”等新名称，不覆盖旧文件。
- 批量生图进行中暂时禁用输出，完成或停止后恢复；输出完成会在访达中定位文件夹。
- 完整 146 项测试通过。
