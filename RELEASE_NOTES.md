# 澄稿 1.21.5

本版本让素材改写页的 AI 控制器明确展示并切换网页 AI 渠道。

## 本版本说明

- 网页 AI 已启用时，控制器主标签直接显示“千问／DeepSeek · 网页 AI”，不再继续显示 API 提供商和模型造成误解。
- 控制器菜单新增“网页 AI”分组，列出千问、DeepSeek、当前启用状态和“停用网页 AI，切回 API”。
- 选择任一 API 模型会自动停用网页 AI；网页模式下停用只适用于 API 请求的推理深度选项。
- 完整 164 项测试通过；安装版辅助功能树实际读取到“千问 · 网页 AI”、DeepSeek 登录入口、切回 API 和 API 模型分组。

公开源码不代表提供经过 Apple 公证的安装包；发布 DMG 前仍需完成签名、公证和依赖许可核验。
