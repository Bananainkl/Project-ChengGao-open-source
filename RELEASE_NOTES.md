# 澄稿 1.20.0

本版本为“新建文稿”增加完整的画面风格选择，并把风格约束贯穿分镜提示词、历史记录、图片生成和 ChatGPT Markdown 导出。

## 本版本说明

- 新增自动匹配、电影纪实、温暖日系手绘动画、经典欧美童话动画、复古美式印刷漫画、夸张追逐喜剧动画、毛线布偶、粘土定格动画和手绘漫画九种风格。
- 每种风格均拆解为材质、造型、光线、色彩和禁止项；使用描述性语言，不直接模仿具体工作室、作品、品牌或角色。
- 选定风格会跨启动保存并写入文稿历史；旧历史没有风格字段时安全回退到“自动匹配”。
- 在线 AI 返回提示词后仍由程序追加全局风格约束，避免不同镜头逐渐漂移；ChatGPT 批量生图文档同步列明统一风格规范。
- 完整 136 项测试通过。

公开源码不代表提供经过 Apple 公证的安装包；发布 DMG 前仍需完成签名、公证和依赖许可核验。
