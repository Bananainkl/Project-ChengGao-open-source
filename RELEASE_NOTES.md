# 澄稿 1.21.1

本版本补齐抖音双封面，并增强不同账号和地区页面下的抖音搜索兼容性。

## 本版本说明

- 配图建议根据标题、完整成稿和画面风格生成 3:4 竖版与 16:9 横版抖音封面提示词。
- 图片模型只生成无字背景，App 在本机排入准确标题并输出 1080×1440／1600×900 PNG。
- 双封面已接入单独生成、批量生成、历史保存、ChatGPT Markdown 与完整文件包。
- 抖音搜索兼容带安全前缀的 JSON、`videoIdStr` 等新版字段，并增加结果卡片后备解析。
- 完整 152 项测试通过；安装版以“职场内耗”真实搜索取得 9 条抖音作品。

公开源码不代表提供经过 Apple 公证的安装包；发布 DMG 前仍需完成签名、公证和依赖许可核验。
