# 澄稿 1.21.6

本版修复短抖音视频与 DeepSeek 网页 AI 的连续处理问题。

- 十几秒视频按真实时长判断有效文字量，不再被固定 40 字门槛误判。
- 音轨没有可识别对白时，使用 macOS Vision 从视频帧读取真实标题和字幕，并过滤重复及竖排商品字样。
- 兼容 DeepSeek 当前无文字发送按钮，避免网页任务提交时报“数据格式不正确”。
- 镜头设计网页请求超过 60 秒或失败时使用本机模板完成，不丢失已经完成的正文。
- 完整 166 项测试通过；安装版真实短视频、DeepSeek 正文改写、镜头后备和结果保存链路通过。

当前安装包为 Apple Silicon（arm64）本地候选包，使用临时签名，尚未经过 Apple Developer ID 公证。
