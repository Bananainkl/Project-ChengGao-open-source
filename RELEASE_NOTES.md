# 澄稿 1.20.2

本版本补齐图片生成接口的独立凭证和远程模型读取，并修复 Key 可能误填进普通地址栏的安全问题。

## 本版本说明

- 图片 API Key 与聊天 API Key 分开保存到不同的 0600 本地凭证文件，修改或删除其中一个不会影响另一个。
- 图片设置新增“读取图片模型”，只使用图片接口地址和图片 Key 获取、缓存远程模型；不再混用聊天模型目录。
- 实际图片生成只读取独立图片 Key，不再隐式复用聊天 Key。
- 图片地址栏会识别并拒绝疑似 API Key；升级时会移除旧版误存在普通设置中的错误字段，并提醒撤销旧 Key 后换新。
- 使用说明补充独立图片 Key、远程模型读取、费用边界和误泄露后的处理方法。
- 完整 144 项测试通过。

公开源码不代表提供经过 Apple 公证的安装包；发布 DMG 前仍需完成签名、公证和依赖许可核验。
