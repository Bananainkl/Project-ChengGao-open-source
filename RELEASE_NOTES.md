# 澄稿 1.20.4

本版本把短视频口播稿统一整理成字幕式的一句话一行，减少导入剪辑软件后重新拆句的工作。

## 本版本说明

- 短视频结果页的第三页改为“字幕式口播稿”，每个完整句子独占一行。
- 复制本页、复制全部、手动编辑保存和旧历史重新打开都使用同一套确定性断句。
- 保护小数／版本号、中文引号和常见英文缩写，公众号、小红书与视频号继续保留段落排版。
- 输出文件包新增纯文本 `口播字幕.txt`，可直接交给剪映等剪辑软件；`改写文稿.md` 同步采用一句一行正文。
- 完整 148 项测试通过。

公开源码不代表提供经过 Apple 公证的安装包；发布 DMG 前仍需完成签名、公证和依赖许可核验。
