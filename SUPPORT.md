# Support

- Use **GitHub Discussions** for usage questions, ideas, and show-and-tell.
- Use the **bug report** issue form for reproducible defects.
- Use the **feature request** form for scoped product proposals.
- Follow `SECURITY.md` for vulnerabilities; never publish sensitive details in
  an issue.

This is a community-maintained open-source project. Response times are not
guaranteed, but clear reproduction steps and small test cases are appreciated.
